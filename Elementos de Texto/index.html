Guia de Elementos HTML Básicos

1. Cabeçalhos (Heading Elements: H1 a H6)

Os elementos `h1` a `h6` são usados para definir títulos de seções e subseções. `h1` é o mais importante (o título principal da página) e `h6` o de menor importância, criando uma hierarquia estrutural.

2. Parágrafo (`p`)

O elemento `p` é utilizado para agrupar blocos de texto, formando parágrafos. O navegador automaticamente adiciona algum espaço antes e depois de cada parágrafo (o que é visível com o CSS aplicado).

Este é o segundo parágrafo de exemplo. Ele demonstra como o navegador renderiza o espaçamento entre dois elementos `p` consecutivos. Use-o para conteúdo textual principal.

3. Listas Não Ordenadas (`ul` e `li`)

O elemento `ul` (Lista Não Ordenada) é usado para agrupar itens que não precisam de uma ordem sequencial específica. Cada item da lista é representado pelo elemento `li` (List Item).

  • Item Não Ordenado A
  • Item Não Ordenado B
  • Item Não Ordenado C

4. Listas Ordenadas (`ol` e `li`)

O elemento `ol` (Lista Ordenada) é usado para agrupar itens que estão em uma sequência ou ordem específica (como um passo-a-passo ou ranking). Assim como na `ul`, cada item é um `li`.

5. Links / Âncoras (`a`)

O elemento `a` (âncora) é usado para criar hiperlinks. O atributo essencial é o `href`, que especifica o destino do link.
